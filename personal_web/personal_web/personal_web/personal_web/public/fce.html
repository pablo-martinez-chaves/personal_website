Fundamentals of Computer Engineering

Description

This course introduces the fundamental concepts of computer engineering, including hardware, software, and computational systems.

Objectives

  • Understand the basic principles of computer science
  • Know the structure of a computational system
  • Introduce logical and computational thinking

Learning Outcomes

  • Identify the main components of a computer
  • Understand the relationship between hardware and software
  • Apply basic concepts of computer engineering
